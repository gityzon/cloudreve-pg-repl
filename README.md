1. Creat a Bash-language project.

   `git clone https://github.com/gityzon/cloudreve-repl && mv -b cloudreve-repl/* ./ && mv -b cloudreve-repl/.[^.]* ./ && rm -rf *~ && rm -rf cloudreve-repl`

3. After "Loading Nix environment..." is done, ▶RUN!!!
